## Installation

- Install globally:

```sh
yarn global add escapin
```

or

```sh
npm install -g escapin
```

## Usage

```sh
escapin [options]
```

- Options

```sh
  -V, --version    output the version number
  -d, --dir <dir>  Working directory
  -h, --help       output usage information
```
